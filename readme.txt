== Description ==

Notifies when the difference between the blog last modified date and the current date exceeds a certain period.

= Notification email =
* Can specify the difference between the last update date and the current date to notify.
* It checks automatically once a day.
* Can change the content of the notification.
* Can notify by specifying a time.

== Installation ==

1. Upload `notify-old-blog` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
